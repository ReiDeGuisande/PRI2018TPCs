Resumo

Este trabalho é referente à criação de um servidor Javascript, que seja capaz de carregar vários ficheiros JSON , e a partir deste conjunto de ficheiros, gerar um indíce, em PUG. A partir desse indíce, geram-se páginas HTML, carregando o ficheiro JSON desejado. Este é lido no servidor, passa por uma transformação em PUG, resultando numa página HTML. Os pedidos de acesso às páginas do website, são controlados pelo servidor. Este website é referente ao Arquivo de Música Digital.
