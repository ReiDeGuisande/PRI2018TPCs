Resumo

Este trabalho é referente à criação de uma aplicação Web, que seja capaz de fazer o upload de ficheiros e listagem destes. Para tal, foi criado um servidor em Javascript, que fosse capaz de processar os pedidos GET e POST. Estes pedidos eram gerados através de Ajax e JQuery. O formulário de submissão de ficheiros, e a sua listagem é feita recorrendo a PUG. Os dados são processados no servidor, e são posteriormente guardados num ficheiro JSON, que servirá como base de dados, para este exercício.
