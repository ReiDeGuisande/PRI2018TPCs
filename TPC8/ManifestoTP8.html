Resumo

Este trabalho é referente à criação de uma aplicação Web, que seja capaz de registar teses e lista-las. Para tal, foi criado um servidor em Javascript, que fosse capaz de processar os pedidos GET e POST. O processamento da informação é feito com o auxílio de formulários, que foram implementados em PUG. Os dados são processados no servidor, e são posteriormente guardados num ficheiro JSON, que servirá como base de dados, para este exercício.
