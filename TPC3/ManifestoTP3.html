Resumo

Este trabalho é referente à criação de uma Stylesheet XSLT, que seja capaz de, a partir de um ficheiro XML fornecido, o quran.xml, criar um conjunto de páginas HTML. Cada página HTML corresponde a um sura e cada um destes suras está identificado na página index, juntamente com mais alguma meta-informação acerca do livro sagrado.

Resultados:

Stylesheet para conversão do Corão.
